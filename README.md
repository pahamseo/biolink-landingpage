# Tokoseo Landing Page

Landing page HTML generik untuk UMKM Indonesia  
Gratis, ringan, dan mudah diduplikasi.

## Cara Pakai
1. Download atau clone repository
2. Buka `index.html`
3. Ganti placeholder seperti:
   - SITE_NAME
   - SITE_URL
   - CTA_TEXT
4. Upload ke hosting

- UMKM
- Edukasi digital
- Komunitas belajar
